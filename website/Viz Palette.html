Color Report

Info

Born out of a frustration with picking colors for data visualizations. This project is optimized for tweaking, copying, and pasting in and out of JavaScript.

Made by Susie Lu & Elijah Meeks

This project relies heavily on Chroma.js a color library made by Gregor Aisch.

The color report uses just-noticeable difference research implemented in d3-jnd by Conor Gramazio as well as color-namer by Zeke Sikelianos.
